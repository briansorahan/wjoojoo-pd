# wjoojoo-pd  

### My personal collection of Pd patches.

- ad~.pd  
    Attack-Decay envelope.

- adsr~.pd  
    Attack-Decay-Sustain-Release envelope.

- delayngin~.pd  
    Delay module.

- fm_osc~.pd  
    Fm oscillator.

- fm_synth_voice~.pd  
    Monophonic FM Synth.

- frq.pd  
    Samples-to-hz converter.

- gain~.pd  
    Input gain with 10ms lag.

- gate~.pd  
    Audio gate.

- lfongin~.pd  
    LFO.

- rate~.pd  
    Implements the MaxMSP rate~ object.

- ringmod_osc~.pd  
    Ringmod oscillator.

- sampngin~.pd  
    Simple sampler engine.

- xfade~.pd  
    Crossfade.
